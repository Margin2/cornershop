# cornershop
街角小店forWechat微信商城

## cornershop特性
1. 无需登录。
2. 包含会员卡、卡券营销等方案。
3. 界面简洁，使用weui，融合微信，无需过度。
4. 区别于微店、乐外卖等第三方微信商城平台，完美契合微信，活动、回款自由可控。、
5. 一次加载无需网页刷新。
6. 延时同步，页面流畅。
7. 安全。直接调用微信支付接口，无第三方。

## 程序配置入口
1. /cornershop/src/iuv/cns/utils/Constants.java
2. /cornershop/resources/jdbc.properties
